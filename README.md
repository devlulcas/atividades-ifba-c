## 👨🏻‍💻 COMO RODAR 👨🏻‍💻

### 🐧 LINUX 🐧
- Instale o GCC, um compilador para a linguagem C:

Arch:
```bash
sudo pacman -S gcc
```
Ubuntu:
```bash
sudo apt install gcc
```

Solus:
```bash
sudo eopkg install -c system.devel
```

- Agora compile o arquivo C:

```bash
gcc arquivo.c -o programa
```

- Agora basta executar o programa:

```bash
./programa
```
